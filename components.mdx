# Clear blocks, easy reading

Explorer Elite pages use a few simple building blocks so readers can scan, compare, and move forward without confusion.

<Callout kind="info">
Use these blocks to help Explorers and Partners find the right next step quickly.
</Callout>

## The main blocks

<Card title="Highlights" icon="sparkles" href="/components/cards" horizontal>
  Use cards for quick choices, featured journeys, and next-step links.
</Card>

<Card title="Guided checklists" icon="list-checks" href="/components/steps" horizontal>
  Use steps when people should move through a journey in order.
</Card>

<Card title="Helpful alerts" icon="message-square" href="/components/callouts" horizontal>
  Use callouts for reminders, tips, and important notes.
</Card>

<Card title="Side-by-side paths" icon="columns-2" href="/components/tabs" horizontal>
  Use tabs when Explorers and Partners need separate views.
</Card>

## When to use what

<Tabs>
  <Tab title="Explorers">
    Explorers often move from introduction to quick answers, then into the pages that help them explore opportunities and understand their Passport.

    Use cards when you want to highlight the next thing to read. Use steps when the reader should do something in order.
  </Tab>

  <Tab title="Partners">
    Partners often want a clear path from overview to setup, then into guides and support.

    Use tabs when the message changes by role. Use callouts when a reminder or expectation matters before someone continues.
  </Tab>
</Tabs>

## Keep the flow simple

- Use cards for choices and related pages.
- Use steps for guided journeys.
- Use callouts for short reminders.
- Use tabs when Explorers and Partners need different guidance.
